Weather Data API Documentation

This API provides access to historical weather station data.

1. Get Temperature for a Specific Station and Date

Endpoint: /api/v1/<station>/<date>/

Example: /api/v1/1/1860-05-12/

Returns the temperature recorded at a station on a specific date.

2. Get All Data for a Station

Endpoint: /api/v1/<station>/

Example: /api/v1/10/

Returns all available weather records for a station.

3. Get Yearly Data for a Station

Endpoint: /api/v1/yearly/<station>/<year>/

Example: /api/v1/yearly/10/1860/

Returns all weather data for a station filtered by year.
